# Repository Instructions

Read `CLAUDE.md` before making code changes. It is the source of truth for this
project's architecture, parity requirements, commands, and working conventions.

## Commit Messages

Do not add any AI attribution, assistant attribution, or co-author trailer to git
commit messages for this repository.

This prohibition includes, but is not limited to:

- `Co-authored-by: Codex <...>`
- `Co-authored-by: OpenAI <...>`
- `Generated with Codex`
- `Generated by ChatGPT`
- `Authored with AI`
- any similar footer, trailer, signature, attribution line, or marketing text
  implying that Codex, ChatGPT, OpenAI, or another AI assistant authored or
  co-authored the commit.

When creating commits, write only the user-requested commit subject and body. If a
template, tool, hook, or default behavior inserts AI attribution automatically,
remove it before committing.
